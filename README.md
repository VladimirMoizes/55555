# Astro Starter Template

Готовая заготовка для многостраничного сайта на Astro с предустановленной структурой и настройками.

## Структура проекта
```
public/
└── assets/          # Шрифты и статические файлы

src/
├── components/      # Переиспользуемые компоненты (Header, Footer)
├── layouts/         # Обертки для страниц
├── pages/           # HTML страницы сайта
└── styles/          # Стили
    ├── variables.css # CSS переменные
    ├── fonts.css     # Подключение шрифтов
    └── global.css    # Глобальные стили
```

## Быстрый старт

```bash
npm install
npm run dev
